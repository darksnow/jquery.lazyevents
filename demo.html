Demo of jquery.lazyevents

This demo page will set up the lazy events on scroll and resize.

It will print a message to the browser console when the event fires so you can see the frequency. If you wish to compare this to the frequency of the normal events, check the "show real events" box and the demo will also show a message for the real events. The real events fire often so may overwhelm the lazy events in the console.

In order to fire the events you'll need to resize or scroll the window (hence the lorum ipsum to create scrollbars for you)
